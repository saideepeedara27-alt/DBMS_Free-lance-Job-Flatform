# Freelance Job Platform — DBMS Project

A MySQL database system to manage freelancers, clients, job postings, proposals, contracts, payments, and feedback.

**University:** NIIT University | **Course:** CS231 DBMS | **Year:** 2024–25

---

## Features

- Relational schema with primary keys, foreign keys, and CHECK constraints
- Views, Triggers, Stored Procedures, and Indexes
- Nested queries and aggregation with GROUP BY / HAVING
